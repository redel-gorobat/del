# üöÄ QUICK SETUP GUIDE

## Step 1: Extract Files
Extract all files from the portfolio folder to your desired location.

## Step 2: Add Your Profile Picture
1. Get your profile photo (350x350px recommended)
2. Save it as `profile.jpg` in the `images/` folder
3. If the image doesn't show, check that the file name matches exactly

## Step 3: Add Project Images

### UI/UX Designs (5 images needed)
Save as: `images/projects/uiux/project1.jpg` through `project5.jpg`

### T-shirt Designs (2 images needed)
Save as: `images/projects/tshirt/design1.jpg` and `design2.jpg`

### Graphic Designs (1 image needed)
Save as: `images/projects/graphic/design1.jpg`

### Logo Designs (3 images needed)
Save as: `images/projects/logo/logo1.jpg` through `logo3.jpg`

## Step 4: Test Locally
1. Open `index.html` in your web browser
2. Navigate through all pages to ensure everything works
3. Add project titles by typing in the input boxes under each image

## Step 5: Add Certificates (Optional)
Place certificate images in the appropriate folders under `images/certificates/`:
- Technical: IT, Programming, Development
- Design: Graphic Design, UI/UX
- Academic: School awards, honors
- Training: Workshops, seminars
- Work: Employment certificates
- Other: Miscellaneous achievements

## Step 6: Customize Social Links
Open `contact.html` and replace `#` with your actual social media URLs:
- Facebook: Line 83
- Instagram: Line 86
- Telegram: Line 89
- Discord: Line 92

## Step 7: Deploy Online
Choose one:

### GitHub Pages (Easiest)
1. Create a free GitHub account
2. Create a new repository
3. Upload all files
4. Go to Settings > Pages
5. Enable GitHub Pages
6. Your site is live!

### Netlify (Fastest)
1. Go to netlify.com
2. Drag and drop your folder
3. Done!

### Vercel
1. Go to vercel.com
2. Sign up and import
3. Deploy

## üé® Customization Tips

### Change Colors
Edit `styles.css` lines 3-15 to change the color scheme

### Add More Pages
1. Copy any existing HTML page
2. Update the content
3. Add link in navigation (lines 9-17 in each HTML file)

### Change Fonts
1. Go to Google Fonts (fonts.google.com)
2. Select your fonts
3. Replace the font link in each HTML file (line 7)
4. Update CSS variables (lines 18-19 in styles.css)

## ‚ùì Common Issues

- Ensure images are in correct folders
